# 闭环最终版V3自查

## 满足项

```text
1.不生成CubeMX可自动生成初始化代码；
2.使用TIM8四路互补PWM；
3.支持Dithering；
4.使用单极性SPWM；
5.包含ADC DMA用户层采样；
6.包含Ud、Id、uREF、iF、uo换算；
7.包含MPPT电导增量法；
8.包含PV电压外环PI；
9.包含PR电流环；
10.包含ADC预热、软启动和保护状态机；
11.修正了V2中滤波初值可能导致启动欠压误故障的问题；
12.修正了V2中Start可从IDLE直接启动的问题；
13.修正了V2中ISR内完整HAL Stop的问题。
```

## 必须由实物确认项

```text
1.PVINV_CFG_TIM8_DITHERING_SCALE是否为16；
2.TIM8更新中断频率是否等于诊断值；
3.ADC Rank顺序是否与宏一致；
4.Ud/Id/uREF/iF/uo比例系数是否标定正确；
5.电流方向符号是否正确；
6.PR参数是否需要根据硬件调整；
7.MPPT步长和范围是否适配真实PV输入。
```

## 不应声称

```text
1.不应声称可直接接高压功率桥；
2.不应声称默认比例系数就是最终硬件比例；
3.不应声称无需示波器和ADC raw标定。
```
